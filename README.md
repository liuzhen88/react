# react project
使用React、Redux、Webpack、Babel、ES6、Webpack、redux-thunk按development和production环境配置好了有关热更新、postcss、hash命名（缓存方案）、UglifyJs等config。

### INSTALL:
```npm install```
### DEVELOPMENT:
```npm start```

``` open http://localhost:8080/ ```

### PRODUCTION:
```npm run build```
